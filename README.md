# crawl-fishpond

## Usage

```
usage: fishpond.py [-h] [--thread THREAD] input output daun_column_num parcel_column_num county town

positional arguments:
  input              Path to input csv file
  output             Path to output json file
  daun_column_num    Daun column number
  parcel_column_num  Parcel column number
  county             County of the csv
  town               Town of the csv

optional arguments:
  -h, --help         show this help message and exit
  --thread THREAD    Number of threads
```
